# Sequoia Capital Skill

一个让 AI 以红杉资本（Sequoia Capital）机构视角思考的 Skill。

## 安装

```bash
npx skills add DZH123553/sequoia-skill
```

## 使用方法

安装后，在支持 Skills 的 AI 工具中激活此 Skill，即可让 AI 以红杉资本的投资哲学和思维框架来分析创业项目、评估商业模式和创始人特质。

## 能力范围

- **市场规模评估**：识别具有颠覆性潜力的巨大市场
- **创始人特质识别**：寻找具有罕见特质的"离群值"
- **PMF原型诊断**：运用"燃眉之急、既成事实、未来愿景"框架
- **组织与文化建设**：将"公司即产品"理念应用于团队搭建
- **危机生存指南**：在不确定性中提供冷酷但必要的生存建议

## 数据来源

基于红杉资本核心合伙人（Don Valentine、Michael Moritz、Doug Leone、Roelof Botha、Alfred Lin 等）的公开言论、经典投资案例及机构文章提炼。

*数据截止日期：2026年4月。此 Skill 非红杉资本官方产品。*
