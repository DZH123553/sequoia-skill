# Sequoia Ultimate Agent

## Description
A highly distilled, cross-platform compatible agent skill that embodies the investment philosophy, mental models, and decision heuristics of Sequoia Capital. Built using the "Anyone to Skill" methodology with rigorous triple-verification.

## Core Components
- **Knowledge Graph**: Located in `references/sequoia_knowledge_graph.md`. Contains the distilled essence of Sequoia's thinking across 5 dimensions (Mental Models, Decision Heuristics, Expression DNA, Internal Tensions, Honesty Boundaries).
- **Execution Script**: Located in `scripts/evaluate_startup.py`. A Python script that applies the knowledge graph to evaluate any given startup or business plan.

## Usage
To evaluate a startup, run the execution script with the startup's description or business plan as input. The agent will analyze the startup through the lens of Sequoia's distilled knowledge graph and provide a structured evaluation.

### Example
```bash
python3 scripts/evaluate_startup.py --input "path/to/startup_pitch.md"
```

## Methodology
This skill was built by analyzing a massive corpus of Sequoia's official articles, partner interviews, leaked memos, and failure post-mortems. The extracted insights were subjected to a "Triple Verification" process to ensure cross-domain recurrence, generative power, and exclusivity.
