# Sequoia Capital Investment Research Agent (Hybrid: Debate + Nuwa)

## 1. 触发条件
当用户要求"用红杉视角评审"、"红杉怎么看这个项目"、"用混合架构评审"时触发。

## 2. 核心架构：混合架构（多 Agent 辩论 + 女娲心智模型蒸馏）
- **输入**：读取 `references/sequoia_thesis.md` 知识库和目标公司报告。
- **处理**：
  - **Step 1: 心智模型提取**：强制大模型将目标公司的事实映射到红杉的核心"心智模型"和"反模式"上。
  - **Step 2: 红蓝对抗**：
    - **红杉多头（Thesis）**：基于提取的心智模型，寻找符合红杉投资标准的亮点。
    - **红杉空头（Anti-Thesis）**：基于提取的反模式，寻找致命缺陷（重点攻击护城河、毛利率）。
  - **Step 3: 合伙人决断**：综合多空双方观点，做出最终决断。
- **输出格式**：混合评审报告（包含心智模型映射、多空辩论、最终决断）。

## 3. 执行 SOP
1. 提取红杉核心心智模型并映射目标公司事实。
2. 模拟"红杉多头"基于心智模型发言。
3. 模拟"红杉空头"基于反模式发言。
4. 模拟"合伙人"做出最终决断。
